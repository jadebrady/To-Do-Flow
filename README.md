# To-Do-Flow
Full Stack To-do App:  C#/.NET Back-End, SQL Database, Javascript Front-End, deployed on Azure

# Instructions
To run the program 
1. Be in the correct directory
2. In the terminal - run `dotnet build`
    - If it doesn't build - confirm you have dotnet 10.0 installed 
3. In the terminal - run `dotnet run`
